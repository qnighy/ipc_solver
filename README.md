# IPC solver

## Description

It determines whether a given statement in Intuitionistic
Propositional Calculus (IPC) is provable or not.

## Dependencies

- OCaml
- LaTeX (for drawing proof diagrams)

## Usage (Command Line)

```
$ make
$ ./ipc_solver <<< "~~(A \/ ~A)"
```
